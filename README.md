# نظام إدارة الصيدلية | Pharmacy Management System

نظام ويب متكامل لإدارة الصيدليات مبني على إطار العمل Laravel.

## 📋 نظرة عامة

نظام احترافي يوفر للصيدليات حلولاً شاملة لإدارة العمليات اليومية بما في ذلك إدارة المنتجات والطلبات والفواتير والموردين والمستخدمين مع واجهة نقطة بيع (POS) حديثة وسهلة الاستخدام.

## ⚙️ المتطلبات التقنية

| المتطلب | الإصدار المطلوب |
|---------|-----------------|
| PHP | 7.3 أو أعلى |
| MySQL | 5.7 أو أعلى |
| Composer | أحدث إصدار |
| Node.js | 14 أو أعلى |

## 🛠️ التقنيات المستخدمة

- **Backend:** Laravel 8
- **قاعدة البيانات:** MySQL
- **الواجهة الأمامية:** Blade Templates, Bootstrap 4
- **الأدوات المساعدة:** DataTables, Select2, Chart.js

## ✨ الميزات الرئيسية

### 🔐 نظام المصادقة والأمان
- نظام تسجيل الدخول والتسجيل الآمن
- إعادة تعيين كلمة المرور
- نظام صلاحيات متقدم (Admin, Cashier, Manager)
- حماية البيانات والتحكم في الوصول

### 🛒 نظام نقطة البيع (POS)
- إنشاء الطلبات بسرعة وكفاءة
- إدارة المنتجات والكميات والخصومات
- طباعة الفواتير الاحترافية
- حساب الإجماليات والضرائب تلقائياً

### 📦 إدارة المخزون
- إضافة وتعديل وحذف المنتجات
- رفع صور المنتجات بسهولة
- نظام تنبيهات عند نقص المخزون
- تتبع الموردين وأسعار الشراء والبيع
- إدارة التصنيفات والأصناف

### 📋 إدارة الطلبات
- عرض جميع الطلبات والفواتير
- تفاصيل شاملة لكل طلب
- البحث والتصفية المتقدم
- تصدير التقارير

### 👥 إدارة المستخدمين
- إدارة حسابات الموظفين
- التحكم في الصلاحيات حسب الدور
- تتبع نشاط المستخدمين

### 📊 التقارير والتحليلات
- تقارير المبيعات اليومية والشهرية
- تقارير المنتجات الأكثر مبيعاً
- تقارير الموردين والعملاء
- رسوم بيانية تفاعلية

### 🏢 إدارة الشركات والموردين
- سجل شامل للموردين
- إدارة بيانات الشركات
- تتبع المعاملات المالية

## 🚀 خطوات التثبيت

1. **نسخ ملف البيئة:**
   ```bash
   cp .env.example .env
   ```

2. **توليد مفتاح التطبيق:**
   ```bash
   php artisan key:generate
   ```

3. **إعداد قاعدة البيانات:**
   عدّل ملف `.env` بإعدادات قاعدة البيانات الخاصة بك:
   ```env
   DB_DATABASE=pharmacy
   DB_USERNAME=root
   DB_PASSWORD=your_password
   ```

4. **تشغيل الهجرات:**
   ```bash
   php artisan migrate
   ```

5. **تثبيت الحزم والاعتماديات:**
   ```bash
   composer install
   npm install && npm run dev
   ```

6. **تشغيل الخادم:**
   ```bash
   php artisan serve
   ```

7. **فتح التطبيق:**
   افتح المتصفح على الرابط: `http://localhost:8000`

## 🌐 المسارات الرئيسية

| المسار | الوصف |
|-------|-------|
| `/` | صفحة تسجيل الدخول |
| `/dashboard` | لوحة التحكم الرئيسية |
| `/orders` | إدارة الطلبات |
| `/products` | إدارة المنتجات |
| `/users` | إدارة المستخدمين |
| `/suppliers` | إدارة الموردين |
| `/companies` | إدارة الشركات |
| `/report` | التقارير والإحصائيات |
| `/pos` | نقطة البيع |

## 🗄️ جداول قاعدة البيانات

- **users** - معلومات المستخدمين والصلاحيات
- **orders** - سجل الطلبات والمبيعات
- **order_details** - تفاصيل المنتجات في كل طلب
- **products** - معلومات المنتجات والمخزون
- **suppliers** - بيانات الموردين
- **companies** - معلومات الشركات
- **transactions** - سجل المعاملات المالية
